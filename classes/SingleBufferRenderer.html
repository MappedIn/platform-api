SingleBufferRenderer Class

Private

Module: Mappedin

A legacy renderer that renders all the elements in the scene to a single buffer. Does not support things like transparent paths behind polygons natively, but can potentially fake it.

2019/07/10 Terence Dickson Generally has worse performance under average loads compared to the multi-buffer renderer, as the multi-buffer renderer can avoid re-rendering the entire map if only animated elements on the map have changed.
