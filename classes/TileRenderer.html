TileRenderer Class

Private

Module: Mappedin

A class that holds the WebGL vertex buffers and shaders used for rendering the Mapbox outdoor context to a WebGL rendering context.

_computeCameraMatrices () private

Determine the camera matrices that will be used for rendering vertex buffers that are in Web SDK coordinates and mapbox Mercator projection coordinates.

_getPosMatrix () private

Compute the model-to-world matrix for a tile, taking coordinates from a Mapbox vertex buffer and transforming them to the SDK world-space coordinates.

This is copied from the computation for posMatrix in the Transform class, with minor edits as mentioned in the comments.

setShouldDrawTiles
(
  • shouldDrawTiles
)

If true, the tiles in the outdoor context will be drawn. If false, only the background and the overlay will be drawn, making this suitable for use in a multi-floor view.

Parameters:

    if true, tiles will be drawn
