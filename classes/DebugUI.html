reportFramePerformance
(
  • entries
)

Log the given entries to this performance logger, and update the display accordingly. If a given key isn't already being logged, an entry will be added for it. If a key is being logged but isn't reported, it will be treated as if it had a value of 0 reported.

Parameters:

  • entries String -> number

    the most recent recorded values for each named frame update
